# Vorpal

> High-performance, functional collection library for TypeScript with lazy evaluation, operation fusion, and early termination. Zero dependencies. Two APIs: Lazy (chainable) and Fn (curried/pipe-friendly).

Install: `npm install @authaz/vorpal-ts`

**Lazy API** - Chainable, deferred execution until terminal operation (.toArray(), .sum(), etc.):
```typescript
import { V } from '@authaz/vorpal-ts/lazy';
V([1, 2, 3, 4, 5])
  .filter(x => x % 2 === 0)
  .map(x => x * 2)
  .take(2)
  .toArray(); // [4, 8]
```

**Fn API** - Curried, data-last functions for pipe composition:
```typescript
import { V } from '@authaz/vorpal-ts/fn';
V.pipe(
  V.filter((x: number) => x % 2 === 0),
  V.map((x: number) => x * 2),
  V.take(2)
)([1, 2, 3, 4, 5]); // [4, 8]
```

**Performance**: Up to 35,000x faster than native arrays for early termination on large datasets. Outperforms Lodash, Ramda, and native in most benchmarks for n > 1,000. For small arrays (n < 100), native methods have less overhead.

**When to use Vorpal**:
- Large arrays (n > 1,000)
- Early termination (take, first, find on large data)
- Chained operations (filter+map+filter+take)
- Aggregations (sum, average, min, max)

**Available operations**: filter, map, flatMap, reduce, take, skip, takeWhile, skipWhile, first, last, find, some, every, includes, count, sum, average, min, max, distinct, groupBy, partition, chunk, sortBy, reverse, concat, zip, union, intersect, difference, innerJoin, leftJoin, and more.
